The improve-aeo-geo skill provides production-ready code patterns for 10 web frameworks. Each pattern covers the five implementation areas that matter most for AEO: metadata (title, description, canonical, Open Graph), JSON-LD structured data, robots.txt with AI crawler allowlisting, sitemap with `<lastmod>` dates, and RSS/Atom feed setup. Copy these patterns directly into your project — they are not pseudocode.

<Tabs>
  <Tab title="Next.js">
    **Metadata** — `app/page.tsx` or `app/layout.tsx`:

    ```typescript
    import type { Metadata } from 'next'

    export const metadata: Metadata = {
      title: 'Page Title | Brand',
      description: 'Descriptive meta description of 120-160 characters.',
      alternates: { canonical: 'https://yourdomain.com/page' },
      openGraph: {
        title: 'Page Title',
        description: 'Social sharing description.',
        url: 'https://yourdomain.com/page',
        images: [{ url: 'https://yourdomain.com/og-image.jpg' }],
      },
      robots: { index: true, follow: true },
    }
    ```

    **JSON-LD** — inline in the page component:

    ```tsx
    export default function Page() {
      const jsonLd = {
        '@context': 'https://schema.org',
        '@type': 'Article',
        headline: 'Article Title',
        datePublished: '2025-01-15',
        dateModified: '2025-06-01',
        author: { '@type': 'Person', name: 'Author Name' },
      }
      return (
        <>
          <script type="application/ld+json" dangerouslySetInnerHTML={{ __html: JSON.stringify(jsonLd) }} />
          <article><h1>Article Title</h1>{/* Content */}</article>
        </>
      )
    }
    ```

    **robots.txt** — `app/robots.ts`:

    ```typescript
    import type { MetadataRoute } from 'next'

    export default function robots(): MetadataRoute.Robots {
      return {
        rules: [
          { userAgent: '*', allow: '/' },
          { userAgent: 'GPTBot', allow: '/' },
          { userAgent: 'ClaudeBot', allow: '/' },
          { userAgent: 'PerplexityBot', allow: '/' },
          { userAgent: 'Google-Extended', allow: '/' },
        ],
        sitemap: 'https://yourdomain.com/sitemap.xml',
      }
    }
    ```

    **Sitemap** — `app/sitemap.ts`:

    ```typescript
    import type { MetadataRoute } from 'next'

    export default function sitemap(): MetadataRoute.Sitemap {
      return [
        { url: 'https://yourdomain.com', lastModified: new Date(), changeFrequency: 'weekly', priority: 1 },
        { url: 'https://yourdomain.com/about', lastModified: new Date(), changeFrequency: 'monthly', priority: 0.8 },
      ]
    }
    ```

  <Tab title="WordPress">
    **functions.php** — Add JSON-LD schema for blog posts:

    ```php
    function add_json_ld_schema() {
      if (is_singular('post')) {
        $schema = [
          '@context' => 'https://schema.org',
          '@type' => 'Article',
          'headline' => get_the_title(),
          'datePublished' => get_the_date('c'),
          'dateModified' => get_the_modified_date('c'),
          'author' => ['@type' => 'Person', 'name' => get_the_author()],
        ];
        echo '<script type="application/ld+json">' . json_encode($schema) . '</script>';
      }
    }
    add_action('wp_head', 'add_json_ld_schema');
    ```

    **robots.txt** — WordPress manages this via Settings → Reading. Add AI bot rules via a plugin (Yoast, Rank Math) or a custom `robots.txt` in the web root.

    **llms.txt** — Create `.well-known/llms.txt` in the WordPress root directory, or use a plugin or rewrite rule to serve it at `/.well-known/llms.txt`.

## robots.txt AI crawler allowlist

Every implementation must explicitly allow all 9 AI crawlers. Blocking any of these crawlers — even by omission if a broad `Disallow: /` rule exists — will prevent AI engines from indexing your content.

```txt
User-agent: *
Allow: /

User-agent: GPTBot
Allow: /

User-agent: ClaudeBot
Allow: /

User-agent: PerplexityBot
Allow: /

User-agent: Google-Extended
Allow: /

User-agent: OAI-SearchBot
Allow: /

User-agent: anthropic-ai
Allow: /

User-agent: ChatGPT-User
Allow: /

User-agent: Bytespider
Allow: /

User-agent: CCBot
Allow: /

Sitemap: https://YOURDOMAIN.com/sitemap.xml
```

The 9 crawlers to explicitly allow: `GPTBot`, `ClaudeBot`, `PerplexityBot`, `Google-Extended`, `OAI-SearchBot`, `anthropic-ai`, `ChatGPT-User`, `Bytespider`, `CCBot`.

<Note>
  If AI bots are blocked in robots.txt, nothing else matters. This is the #1 prerequisite for any AEO improvement. Check your existing robots.txt for broad `Disallow` rules that might inadvertently block these crawlers before implementing anything else.
</Note>
